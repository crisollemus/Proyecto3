La box-shadowpropiedad CSS se usa para aplicar una o más sombras a un elemento. Puedes aplicar las siguientes propiedades:
1. Especificar una sombra horizontal y una vertical.
2. Especificar un color para la sombra.
3. Agregar un efecto de desenfoque a la sombra.
4. Establecer el radio de propagación de la sombra.
5. Agregar múltiples sombras.
